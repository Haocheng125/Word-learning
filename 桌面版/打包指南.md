# 单词学习助手 - 打包为 EXE 指南

## 方式一：直接运行（推荐，最简单）

1. 双击 `启动应用.bat` 即可运行
2. 或者在命令行运行：`python main.py`

## 方式二：打包为 EXE 文件

### 步骤 1：安装依赖

```bash
pip install PyInstaller
pip install PyPDF2 pdfplumber
```

### 步骤 2：打包

在 `背单词` 文件夹下打开命令行，运行：

```bash
pyinstaller --onefile --windowed --name "单词学习助手" main.py
```

或者使用配置文件：

```bash
pyinstaller --clean build.spec
```

### 步骤 3：获取 EXE 文件

打包完成后，EXE 文件位于：
```
dist/单词学习助手.exe
```

### 步骤 4：部署到网站

将 `dist/单词学习助手.exe` 复制到：
```
backend/uploads/downloads/单词学习助手.exe
```

然后用户就可以通过 `/download` 页面下载了！

## PyInstaller 参数说明

- `--onefile`: 打包成单个 EXE 文件
- `--windowed`: 不显示控制台窗口
- `--name`: 指定输出文件名
- `--clean`: 清理临时文件

## 常见问题

### 问题 1：找不到 pyinstaller 命令

解决：使用 `python -m PyInstaller` 代替 `pyinstaller`

### 问题 2：打包后运行报错

解决：检查是否所有依赖都被包含，可以在 build.spec 中添加 hiddenimports

### 问题 3：EXE 文件太大

解决：可以使用 `--onedir` 代替 `--onefile`，生成文件夹形式
